The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content.
5 ways we can select DOM. They are:-
1. getElementById()
2. getElementsByClassName()
3. getElementsByTagName()
4. querySelector()
5. querySelectorAll()
Event delegation refers to the process of using event propagation (bubbles).
To manipulate an element's attributes in the DOM, one can use getAttribute() to retrieve values, setAttribute() to modify or add attributes, and Attribute() to remove attributes. For styling, we can access style properties to directly change CSS properties like color or background color.
